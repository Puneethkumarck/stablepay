Architecture

Engineered end-to-end

On-chain

Solana Escrow Program

Custom Anchor program with deposit, claim, and refund instructions. PDA-based escrow accounts with time-locked expiry. All funds verifiable on Solana Explorer.

Anchor 0.32 Rust USDC SPL
Backend

Spring Boot Orchestrator

Hexagonal architecture with domain-driven design. Temporal workflows orchestrate the full remittance lifecycle with automatic retries and guaranteed delivery.

Java 25 Spring Boot 4 Temporal PostgreSQL
Frontend

Next.js Sender App

Mobile-first web app with dark theme. Server Components for performance, BFF pattern for security. Three-step send wizard with live FX rates and real-time status tracking.

Next.js 16 React 19 TypeScript 6 Tailwind 4
Security

MPC Wallet Abstraction

Threshold EdDSA signing via a Go sidecar. Users never see a private key. Google OAuth login provisions a Solana wallet automatically at sign-up.

Go 1.26 TSS-Lib EdDSA gRPC
Fiat On-Ramp

Stripe Payments

Credit/debit card payments via Stripe Payment Intents. Webhook-confirmed funding deposits USDC into the user's Solana wallet. Min $1, max $10,000.

Recipient Experience

No app. No wallet.
Just a link.

SMS → Browser → UPI

The recipient receives an SMS with a claim link. They open it in any browser — no app download, no wallet setup, no crypto knowledge required.

They see who sent the money, the exact amount in rupees, a countdown timer, and enter their UPI ID to receive INR directly to their bank account within seconds.

On-Chain Proof

Every transfer is verifiable

Real escrow on Solana devnet

Each remittance creates a PDA-based escrow account on Solana. The account holds USDC until claimed or refunded. Every step — deposit, claim, refund — is a signed on-chain transaction visible on Solana Explorer.
